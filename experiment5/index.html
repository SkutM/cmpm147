Description

This project makes geometric shapes evolve to get very near a mosaic of a selected background image using mutation and rendering. You can control the mutation rate and restart the evolutionary process with different references!

Reflection

I learned how to combine mutation strategies with rendering, creating a moving image that blends randomness and control. This project taught me to separate data (initDesign) from drawing, which resulted in an evolving generative artwork!

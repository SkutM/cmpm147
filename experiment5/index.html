Description

This project evolves a set of geometric shapes to approximate a selected background image using mutation and rendering strategies. Users can control the mutation rate and restart the evolutionary process with new inspirations.

Reflection

I learned how to combine evolutionary mutation strategies with visual rendering, creating a system that blends randomness and control. This project taught me to separate data generation from drawing, resulting in an evolving generative artwork.

Experiment 3 - Generative Overworld and Dungeon

Description

In this experiment I made procedural tile worlds using Perlin noise and random walk algorithms. The world can be toggled between a procedural overworld and dungeon, with their own generative logic and decorations!

Technical

The overworld uses Perlin noise for terrain generation. The dungeon is made of randomly placed rooms connected with hallways. Water and trees have dynamic animations using millis() and mouseX.

Reflection

I struggled with canvas rendering order and DOM targeting in setup(), especially when running on GitHub Pages. I did however improve my grasp of 2D tile rendering and procedural logic -- something I haven't really explored yet! The generator can be accessed here: https://glitch.com/edit/#!/polite-brick-shrew

Experiment 3 - Generative Overworld and Dungeon

Description

This experiment explores procedural worldbuilding using Perlin noise and random walk algorithms. The world can be toggled between a natural overworld and a dungeon interior, each with its own generative logic and decor.

Technical

The overworld uses Perlin noise for terrain generation. The dungeon is made of randomly placed rooms connected with hallways. Tiles are placed based on map context, and decorations are probabilistically chosen. Water and trees have dynamic animations using millis() and mouseX.

Reflection

I struggled with canvas rendering order and DOM targeting in setup(), especially when running on GitHub Pages. I learned how to delay canvas creation until elements are rendered. I also improved my grasp of 2D tile-based rendering and procedural logic.

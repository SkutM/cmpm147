Experiment 4 - Infinite Worlds: Cosmic Space

Description

Explore an infinite cosmic world filled with twinkling stars, colorful planets, and dynamic explosions. Clicking generates planets; clicking again triggers a vibrant explosion.

Technical

This project uses p5.js with procedural generation. Every planet and star's position is determined by a key input. It uses per-tile seeding, time-based animations, and mouse click interactions.

Reflection

I learned how to combine random seed hashing and tile-based procedural generation to create a lively, infinite space. I also explored dynamic animations and user interactions to make the world feel alive.

Experiment 4 - Infinite Worlds: Cosmic Space

Description

Explore an infinite cosmic world filled with twinkling stars, colorful planets, and explosions. Clicking generates planets, and clicking again triggers an explosion.

Technical

This project uses p5.js with procedural generation. Every star's position is determined by a world key input. It uses seeds, animations, and mouse click interactions.

Reflection

I learned how to combine random seed hashing and tile procedural generation to create an infinite space. I also made animations and user interactions to make the world feel alive. Due to some technical limitations on GitHub Pages (though it sometimes works if you reload the page enough), you may need to open Inspect Element to activate keyboard and mouse interactions. For the best experience, please view my version on Glitch.

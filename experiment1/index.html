Experiment 1 - Hello Glitch/Generator

Recipe Generator

Description

I have created a grammar-based text generator on Glitch that gives the user... interesting cooking recipes to follow.

Technical

I edited the "fillers" object to include ingredients, utensils, cooking actions, and outcomes instead of the fantasy terms. I also rewrote the "template" string to read like a cooking guide instead of a quest letter.

Reflection

This process was generally smooth! Following the instructions was very simple, and required mostly editing some names and descriptions and setting up Github Pages than anything. Super simple experiment!
